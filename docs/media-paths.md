# Media Paths

| Content | Manager | Download category | Final folder | Plex library |
|---|---|---|---|---|
| Movies | Radarr | `movies` | `X:\Plex\MOVIES` | Movies |
| Documentary movies | Radarr | `movies` | `X:\Plex\Documentary Movies` | Documentary Movies |
| TV series | Sonarr | `tv` | `X:\Plex\TV SERIES` | TV Shows |
| Documentary series | Sonarr | `tv` | `X:\Plex\Documentary Series` | Documentary Series |

Root selection is per title. Sonarr and Radarr do not reliably infer whether a
title is a documentary, so select the correct root when adding it.

- `Gannibal` belongs in `X:\Plex\TV SERIES`.
